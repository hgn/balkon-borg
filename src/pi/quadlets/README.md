# src/pi/quadlets — Podman quadlets

Not started. Will hold the Podman quadlet units for the borg-pi5's backend services:
Mosquitto (MQTT broker), Frigate (camera/object detection), readsb/tar1090 (ADS-B),
BirdNET-Go (bird-call recognition). The RTL-SDR V4 needs a current `librtlsdr`
(osmocom ≥ 2.0.1) for readsb — see
[`../../../docs/build-notes.md`](../../../docs/build-notes.md).
